**Create `src/hooks/useDashboardData.ts`:**

Follow the established pattern from `usePlayerSearch.ts` (cancelled flag, cleanup, error classification).

The hook:
```typescript
export function useDashboardData(): DashboardData
```

Internally calls `useSession()` to get the licence number. Returns:
```typescript
interface DashboardData {
  profile: PlayerProfile | null;
  recentMatches: MatchPreview[];  // Last 3 matches
  quickStats: QuickStats | null;
  isLoading: boolean;       // True on initial load
  isRefreshing: boolean;    // True on pull-to-refresh
  error: string | null;     // i18n key
  refresh: () => Promise<void>;
}

interface MatchPreview {
  date?: string;
  opponent?: string;
  score?: string;
  event?: string;
  round?: string;
  isWin?: boolean;  // Derived from result data if available
}

interface QuickStats {
  bestRanking: { discipline: string; classement: string; cpph: number } | null;
  matchCount: number;
  winRate: number;  // 0-100
}
```

Implementation:
1. Get `session` from `useSession()`. Guard: if no session or no licence, return early with empty state.
2. `fetchData(isRefresh = false)` — Fetches in parallel using `Promise.allSettled([getPlayerProfile(licence), getResultsByLicence(licence)])`:
   - Handle partial success: if profile succeeds but matches fail, show profile data with empty matches.
   - For matches: if `Retour` is an array, map to `MatchPreview[]` (slice last 3 for preview). Map fields from `ResultItem` schema: `Date` -> `date`, `Adversaire` -> `opponent`, `Score` -> `score`, `Epreuve` -> `event`, `Tour` -> `round`.
   - Compute `QuickStats` using `getBestRanking()` from rankings utils. Match count = total array length. Win rate requires parsing results — if we can't determine wins, set winRate to 0.
3. `refresh()` — Calls `fetchData(true)` which sets `isRefreshing` instead of `isLoading`.
4. `useEffect` triggers `fetchData()` on mount with cleanup cancellation pattern.
5. Error handling: Classify errors using `NetworkError`, `ServerError` pattern from existing hooks. Map to i18n keys: `dashboard.networkError`, `dashboard.loadError`.
