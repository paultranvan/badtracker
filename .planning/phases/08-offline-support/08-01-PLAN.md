  <action>
1. Install @react-native-community/netinfo:
   ```
   npx expo install @react-native-community/netinfo
   ```

2. Create `src/cache/storage.ts` — thin cache abstraction over AsyncStorage:
   - Use prefix `badtracker_cache:` for all cache keys (distinct from `badtracker_bookmarks` and `badtracker_language`)
   - `cacheGet<T>(key: string): Promise<T | null>` — reads + JSON.parse, returns null on error/miss
   - `cacheSet<T>(key: string, data: T): Promise<void>` — JSON.stringify + write, silently catches errors (fire-and-forget, matching bookmarks/storage.ts pattern)
   - `cacheClear(): Promise<void>` — getAllKeys, filter by CACHE_PREFIX, multiRemove. MUST NOT touch keys without the prefix (preserves bookmarks + language)
   - `cacheClearForUser(licence: string): Promise<void>` — clears only keys containing the licence (for targeted cleanup)
   - Export the CACHE_PREFIX constant for testing
   - Follow the same error-swallowing pattern as bookmarks/storage.ts (try/catch with empty catch blocks)
   - Key patterns to document in comments:
     - `badtracker_cache:dashboard:{licence}` — dashboard data
     - `badtracker_cache:matches:{licence}` — match history
     - `badtracker_cache:ranking:{licence}` — ranking evolution
     - `badtracker_cache:club:{clubId}` — club leaderboard
     - `badtracker_cache:player:{licence}` — bookmarked player profile

3. Create `src/connectivity/context.tsx` — connectivity provider + offline bar:
   - Import `useNetInfo` from `@react-native-community/netinfo`
   - Create `ConnectivityContext` with `{ isConnected: boolean }` (default `true`)
   - `ConnectivityProvider` component:
     - Uses `useNetInfo()` hook
     - Computes `isConnected`: `netInfo.isConnected !== false && netInfo.isInternetReachable !== false` (treats initial null as connected per research pitfall #1)
     - Tracks previous connectivity with useRef to detect reconnection events
     - Exposes optional `onReconnect` callback prop for auto-refresh on connectivity return
   - `useConnectivity()` hook: standard useContext with guard (throw if outside provider, matching useSession/useBookmarks pattern)
   - `OfflineBar` component:
     - Uses `useConnectivity()` to check `isConnected`
     - Returns `null` when connected (auto-dismisses immediately per user decision)
     - When offline: renders a small View (height ~32px) with orange background (#f97316), centered white text using i18n key `offline.noConnection`
     - No animation needed (auto-dismiss = simply unmount)
  </action>
  <verify>
    - `npx tsc --noEmit` passes (TypeScript compiles)
    - `src/cache/storage.ts` exports cacheGet, cacheSet, cacheClear, cacheClearForUser
    - `src/connectivity/context.tsx` exports ConnectivityProvider, useConnectivity, OfflineBar
    - grep confirms CACHE_PREFIX does NOT match 'badtracker_bookmarks' or 'badtracker_language'
  </verify>
  <done>
