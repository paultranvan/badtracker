<task type="auto">
  <name>Task 1: Build match history screen with all features</name>
  <files>app/(app)/(tabs)/matches.tsx</files>
  <action>
Create the full match history screen as a new tab. Use `useMatchHistory` hook from Plan 01 for all data.

**Screen structure (top to bottom):**

**A. Collapsible Stats Header (Animated)**
- Use `Animated.Value` for scroll tracking: `const scrollY = useRef(new Animated.Value(0)).current`
- Header max height ~140px, collapses to 0px on scroll
- `headerHeight = scrollY.interpolate({ inputRange: [0, 140], outputRange: [140, 0], extrapolate: 'clamp' })`
- `headerOpacity = scrollY.interpolate({ inputRange: [0, 70], outputRange: [1, 0], extrapolate: 'clamp' })`
- Use `useNativeDriver: false` for height animation (height cannot use native driver)
- Content inside: win percentage progress bar + "15V - 8D" text
- Progress bar: colored bar (green portion = win%, remainder = loss background in light red/gray)
- Win percentage text: `t('matchHistory.winRate', { rate: stats.winPercentage })`
- Win/loss counts: `t('matchHistory.wins', { count: stats.wins })` + `t('matchHistory.losses', { count: stats.losses })`
- Stats follow active discipline filter (stats from hook already filtered)
- Wrap in `<Animated.View style={{ height: headerHeight, opacity: headerOpacity, overflow: 'hidden' }}>`

**B. Discipline Filter Chips (fixed, not collapsible)**
- Horizontal row of tappable chips: Tous | Simple | Double | Mixte
- Each chip shows count from disciplineCounts: e.g., "Simple (12)"
- Active chip: blue background (#2563eb), white text
- Inactive chip: gray border (#e5e7eb), dark text
- `onPress` calls setDiscipline(filter)
- Use `ScrollView horizontal` to contain chips (in case of overflow), or just a `View` with `flexDirection: 'row'` and `gap: 8`

**C. Season Picker**
- Simple row below chips with "Saison:" label and horizontal chip-style selector
- Show available seasons from hook (e.g., "2025-2026", "2024-2025")
- Plus an "All" option (sets activeSeason to null)
- Active season chip styled same as discipline chips (blue active, gray inactive)
- If only one season available, still show it but no "All" option needed

**D. SectionList (main content)**
- `<SectionList sections={sections} ...>`
- `renderSectionHeader`: Tournament name in bold + date on right side. Sticky headers enabled (default on iOS, set `stickySectionHeadersEnabled={true}` for Android too).
- `renderItem`: Match row component (see below)
- `keyExtractor`: `(item) => item.id`
- `onScroll`: `Animated.event([{ nativeEvent: { contentOffset: { y: scrollY } } }], { useNativeDriver: false })`
- `scrollEventThrottle={16}` for smooth scroll tracking
- `refreshControl`: `<RefreshControl refreshing={isRefreshing} onRefresh={refresh} colors={['#2563eb']} tintColor="#2563eb" />`
- `ListEmptyComponent`: Show "Aucun match trouvé" or "Aucun match pour ce filtre" based on whether allMatches is empty or just filtered is empty
- `contentContainerStyle={{ paddingBottom: 32 }}` for bottom spacing

**E. Match Row Component (sub-component)**
- Left: W/L badge — circular, V (green background #dcfce7) or D (red background #fee2e2), same style as dashboard MatchRow
- Center column:
  - Opponent name (tappable → `router.push('/player/${match.opponentLicence}')` if licence available, otherwise plain text). Use `<Pressable>` for the tappable name with blue color (#2563eb).
  - For doubles/mixed with partner: show "avec {partner}" in small gray text below opponent
  - For doubles with two opponents: show both opponent names
  - Round text in small gray: e.g., "1/4 finale"
- Right: Score text + discipline icon letter (S/D/M) in small badge
- Entire row is tappable to expand/collapse accordion
- `onPress` calls `toggleMatchExpand(match.id)`

**F. Accordion Expanded Detail (conditional)**
- Show only when `expandedMatchId === match.id`
- LayoutAnimation handles the expand/collapse transition (already set up in hook)
- Content (show only if data available, skip row if undefined):
  - Set scores: "Sets : 21-15, 18-21, 21-12" (join setScores array with ", ")
  - Points impact: "+12.5 pts" (green text for positive, red for negative)
  - Tournament name (tappable — filter to that tournament by calling setDiscipline('all') then filtering; or simpler: just display, the tournament filter from context is a nice-to-have). Show tournament name as tappable blue text.
  - Duration: "Durée : 45 min" if available
- Background: slightly lighter than row (#f9fafb), with left blue border accent
- Padding and clean spacing

**G. Loading and Error States**
- Loading: centered ActivityIndicator (same as dashboard pattern)
- Error: centered error text + retry button (same as dashboard pattern)
- Use i18n keys for all text

**Styles:**
- Follow existing app color palette: #2563eb (primary blue), #374151 (dark text), #6b7280 (secondary text), #9ca3af (light text), #f3f4f6 (borders), #dcfce7 (win green), #fee2e2 (loss red)
- Use StyleSheet.create at bottom of file (not inline styles)
- Badge styles consistent with dashboard MatchRow
