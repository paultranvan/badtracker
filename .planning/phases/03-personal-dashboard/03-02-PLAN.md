  <action>
Replace the entire content of `app/(app)/(tabs)/index.tsx` with a full dashboard screen.

**Imports:**
- React Native: `View`, `Text`, `ScrollView`, `RefreshControl`, `Pressable`, `ActivityIndicator`, `StyleSheet`
- expo-router: `router`
- react-i18next: `useTranslation`
- Custom: `useDashboardData` from `../../../src/hooks/useDashboardData`
- Custom: `getRankGaps`, `getRankLabel` from `../../../src/utils/rankings`
- Custom: `useSession` from `../../../src/auth/context`

**Screen structure (top to bottom in ScrollView):**

1. **Greeting Header:**
   - Show personalized greeting: `t('dashboard.greeting', { name: session.prenom || session.nom })`
   - If the hook indicates a recent win in latest match (isWin on first match), use `t('dashboard.greetingWin', { name })` instead
   - Subtle text, not a banner — part of the scrollable content

2. **Quick Stats Row:**
   - Horizontal row of 2-3 stat cards (use flexDirection: 'row', flex: 1 per card)
   - **Best Ranking:** Show `quickStats.bestRanking.classement` prominently, discipline name below. If null, show "NC"
   - **Matches Played:** Show `quickStats.matchCount` as number, label below
   - **Win Rate:** Show `quickStats.winRate` as percentage (e.g., "75%"), label below
   - Style: Each stat card has a light background (#f0f7ff or similar), rounded corners, centered text

3. **Ranking Cards Section:**
   - Section header: `t('dashboard.rankings')`
   - Row of 3 cards side by side (flexDirection: 'row', flex: 1 each, gap: 8)
   - Each card for Simple, Double, Mixte:
     - Discipline name at top (translated via `t('player.simple')`, etc.)
     - Category label large and bold (e.g., "P11") — use `getRankLabel(ranking?.classement)`
     - CPPH value below: e.g., "15.2 pts"
     - Gap indicators below CPPH:
       - If `toNext` not null: `t('dashboard.pointsToNext', { points: toNext, rank: nextRankName })`
       - If `toPrev` not null: `t('dashboard.pointsAbovePrev', { points: toPrev, rank: prevRankName })`
     - For unranked (NC): show "NC" as category, no CPPH, gap shows distance to P12
     - Card border color: use `#2563eb` (blue) for the best-ranked discipline, `#e5e7eb` for others
   - **Pressable:** Each card wrapped in `Pressable` — onPress navigates as stub for Phase 5:
     ```typescript
     // Stub: Phase 5 will add ranking evolution route
     // For now, navigate to own player profile
     router.push(`/player/${session.licence}`);
     ```

4. **Recent Matches Section:**
   - Section header: `t('dashboard.recentMatches')`
   - If no matches: show `t('dashboard.noMatches')` text
   - For each match (up to 3):
     - Row layout: V/D badge | opponent name | score
     - V/D badge: Colored circle or pill — green background with "V" (victory) or red with "D" (defeat), using `t('dashboard.victory')` / `t('dashboard.defeat')`
     - If `isWin` is undefined, show a neutral gray badge with "?"
     - Opponent name: text, truncated if long
     - Score: right-aligned
     - Optional: event name below in smaller gray text
   - **"Voir tous les matchs" link** at bottom of section:
     ```typescript
     // Stub: Phase 4 will add match history route
     // For now, show the link but navigate to a placeholder or do nothing
     ```
     Style as a pressable text link with blue color and arrow indicator

5. **Pull-to-Refresh:**
   - Wrap entire content in `ScrollView` with `RefreshControl`:
     ```typescript
     <ScrollView
       refreshControl={
         <RefreshControl
           refreshing={isRefreshing}
           onRefresh={refresh}
           colors={['#2563eb']}
           tintColor="#2563eb"
         />
       }
     >
     ```

6. **Loading State:**
   - On initial load (`isLoading` true): Show centered `ActivityIndicator` with `#2563eb` color (same as player profile screen pattern)

7. **Error State:**
   - On error: Show error message centered with retry button (same pattern as player profile screen)
   - Use `t(error)` to display translated error message
   - Retry button calls `refresh()`

**Styling approach:**
- Use `StyleSheet.create` (consistent with codebase)
- Color scheme: `#2563eb` (blue) for accents, `#111` for primary text, `#6b7280` for secondary, `#9ca3af` for tertiary
- Card backgrounds: `#f9fafb` with `#e5e7eb` borders (matches player profile card style)
- Stat card backgrounds: `#eff6ff` (very light blue) for visual emphasis
- Spacing: 16px padding, 12px between sections, 8px between cards
- Font sizes: Greeting 20, stat values 24 bold, card category 20 bold, section headers 18 semibold

All text must use `useTranslation()` — no hardcoded strings. Use the dashboard.* keys from the locale files created in Plan 01.
