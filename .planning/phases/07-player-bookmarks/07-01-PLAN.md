**Step 2: Create `src/bookmarks/storage.ts`**

Thin AsyncStorage wrapper for bookmark persistence. Define:

```typescript
export interface BookmarkedPlayer {
  licence: string;
  nom: string;
  prenom: string;
  rankings: {
    simple?: string;   // classement string e.g. "P11"
    double?: string;
    mixte?: string;
  };
  bookmarkedAt: number; // Date.now() timestamp
}
```

Functions:
- `loadBookmarks(): Promise<BookmarkedPlayer[]>` ﻗ°½ AsyncStorage.getItem + JSON.parse, return `[]` on error
- `saveBookmarks(bookmarks: BookmarkedPlayer[]): Promise<void>` ﻗ°½ AsyncStorage.setItem + JSON.stringify, silent catch (fire-and-forget pattern matching `useClubSearch.ts`)
- `updateBookmarkRankings(licence: string, rankings: BookmarkedPlayer['rankings'], all: BookmarkedPlayer[]): BookmarkedPlayer[]` ﻗ°½ returns new array with updated rankings for the given licence (used for passive ranking refresh when visiting a bookmarked player's profile)

AsyncStorage key: `'badtracker_bookmarks'`

Import AsyncStorage from `@react-native-async-storage/async-storage` (already installed v2.2.0).

**Step 3: Create `src/bookmarks/context.tsx`**

Follow the exact pattern from `src/auth/context.tsx`:
- `BookmarksContext` via `createContext<BookmarksContextType | null>(null)`
- `useBookmarks()` hook that throws if used outside provider
- `BookmarksProvider` component:
  - `useState<BookmarkedPlayer[]>([])` for bookmarks array
  - `useState<boolean>(false)` for `isLoaded`
  - `useEffect` on mount: call `loadBookmarks()`, set state, set `isLoaded = true`
  - `addBookmark(player: BookmarkedPlayer)` via `useCallback`: check for duplicate by licence, append, call `saveBookmarks` (fire-and-forget)
  - `removeBookmark(licence: string)` via `useCallback`: filter out by licence, call `saveBookmarks` (fire-and-forget)
  - `isBookmarked(licence: string)` via `useCallback` with `[bookmarks]` dependency: returns `bookmarks.some(b => b.licence === licence)`
  - `updateStoredRankings(licence: string, rankings: BookmarkedPlayer['rankings'])` via `useCallback`: updates a bookmarked player's rankings snapshot in state and storage (for passive refresh on profile visit)

Context value type:
```typescript
interface BookmarksContextType {
  bookmarks: BookmarkedPlayer[];
  isBookmarked: (licence: string) => boolean;
  addBookmark: (player: BookmarkedPlayer) => Promise<void>;
  removeBookmark: (licence: string) => Promise<void>;
  updateStoredRankings: (licence: string, rankings: BookmarkedPlayer['rankings']) => void;
  isLoaded: boolean;
}
```

Note: Bookmarks persist even after logout (tied to device, not account per user decision). Provider wraps OUTSIDE SessionProvider awareness ﻗ°½ it loads once on app mount regardless of auth state.
